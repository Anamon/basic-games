DELIVER THE CAKE

"Deliver the Cake" is a text adventure involving the delivery of a
cake in an enchanted house.

This game was found in a collection of old TI disk images.

The original image name and path was "0003.dsk/CAKE"

The game is in TI BASIC, was originally written by Furman Smith, and
was published in the July/August 1981 issue of Recreational Computing.
