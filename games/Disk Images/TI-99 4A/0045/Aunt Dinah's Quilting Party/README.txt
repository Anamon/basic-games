AUNT DINAH'S QUILTING PARTY

This is a rendition of "Aunt Dinah's Quilting Party (I was Seeing
Nellie Home)".

This game was found in a collection of old TI disk images.

The original image name and path was "0045.dsk/NELLIE".

The game is in TI Extended BASIC and credited to Jim Peterson, but no
further information is known.
