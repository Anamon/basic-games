HOME IN DREAMS

This program performs a song called "Home in Dreams", written by the
programmer's late brother, Donald W. Peterson.

This game was found in a collection of old TI disk images.

The original image name and path was "0045.dsk/HOMEDREAMS".

The game is in TI Extended BASIC, credited to Jim Peterson and
Tigercub Software, and dedicated to the memory of Donald Peterson, but
no further information is known.
