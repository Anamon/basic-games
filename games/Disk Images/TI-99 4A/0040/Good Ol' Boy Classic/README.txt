A GOOD OL' BOY CLASSIC

This is a compilation of song renditions, including "Gavoti Hoedown".

This game was found in a collection of old TI disk images.

The original image name and path was "0040.dsk/GOODOLDBOY".

The game is in TI Extended BASIC, with credits given to T. Brown, Bill
Palmer, and S. Foster, but no further information is known.
