I'M FOREVER BLOWING BUBBLES

This is a rendition of "I'm Forever Blowing Bubbles".

This game was found in a collection of old TI disk images.

The original image name and path was "0040.dsk/BUBBLEBLOW".

The game is in TI Extended BASIC, with music credited to Earl Dodd and
graphics to Jim Peterson, but no further information is known.
