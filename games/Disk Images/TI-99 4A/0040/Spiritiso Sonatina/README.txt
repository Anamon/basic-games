SPIRITISO SONATINA

This is a rendition of Clementi's "Spiritiso" Sonatina.

This game was found in a collection of old TI disk images.

The original image name and path was "0040.dsk/SONATINA".

The game is in TI BASIC and credited in 1982 to Ron Albright of
Columbus, GA, but no further information is known.
