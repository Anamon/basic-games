SARABANDE

This is a rendition of George Friedrich Handel's "Sarabande", followed
by "The Jolly Blacksmith".

This game was found in a collection of old TI disk images.

The original image name and path was "0040.dsk/SARABANDE".

The game is in TI BASIC and credited to Greg and Kim Cagle of Plant
City, FL, but no further information is known.
