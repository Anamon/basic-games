A WHITER SHADE OF PALE

This is a rendition of Procol Harum's "A Whiter Shade of Pale".

This game was found in a collection of old TI disk images.

The original image name and path was "008b.dsk/WHITERSHOP".

The game is in TI Extended BASIC, and is credited to Tony Knerr, but
no further information is known.
