THE ENTERTAINER

This is a rendition of Scott Joplin's "The Entertainer".

This game was found in a collection of old TI disk images.

The original image name and path was "008a.dsk/ENTERTANER".

The game is in TI Extended BASIC, and is credited in 1986 to Daniel
C. Hawes of Salt Lake City, UT, but no further information is known.
