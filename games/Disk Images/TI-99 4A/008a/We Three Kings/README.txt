[Note: This game requires TI BASIC and will not run in TI Extended
Basic, which allows fewer characters to be redefined.]

WE THREE KINGS

This is a rendition of the Christmas carol "We Three Kings", with a
graphical accompaniment.

This game was found in a collection of old TI disk images.

The original image name and path was "008a.dsk/WE-3-KINGS".

The game is in TI BASIC, but no further information is known.
