TOCCATA AND FUGUE

This is a rendition of J. S. Bach's Toccata and Fugue in D Minor.

This game was found in a collection of old TI disk images.

The original image name and path was "008a.dsk/TOCCAT".

The game is in TI Extended BASIC and credited in August 1983 to Don
Gilchrist of the TI Canberra Home User's Group, but no further
information is known.
