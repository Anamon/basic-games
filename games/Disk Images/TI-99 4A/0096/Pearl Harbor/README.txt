PEARL HARBOR

This is a shoot-'em-up game involving shooting Japanese Zeros from a
submarine at Pearl Harbor, using surface-to-air missiles.  The
historical accuracy is... tentative.

This game was found in a collection of old TI disk images.

The original image name and path was "0096.dsk/HARBOR".

The game is in TI Extended BASIC, and is credited to Jerry and Cindy
Hunter, but no further information is known.
