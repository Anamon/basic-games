TI BOMBER

This is a TI adaptation of the game "Bombrun" from "More BASIC
Computer Games".

This game was found in a collection of old TI disk images.

The original image name and path was "0090.dsk/TIBOMBER".

The game is in TI BASIC, and the adaptation is credit to Milo
Tsukroff, but no further information is known.  The original version
was written by Jim Prelesnik.
