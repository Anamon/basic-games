NOTEWORTHY

Noteworthy is a platform game involving the collection of music notes.
Use S/D to move left and right and the P key to jump.

This game was found in a collection of old TI disk images.

The original image name and path was "0002.dsk/$NOTEWRTH".

The game is in TI Extended BASIC and apparently by R. Trueman, but no
further information is known.
