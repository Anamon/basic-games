ADJECTIVE TO ADVERB V1.3

Adjective to Adverb is an educational lesson in forming the adverb
form of adjectives.

This game was found in a collection of old TI disk images.

The original image name and path was "0025.dsk/ADJ/ADV".

The game is in TI Extended BASIC and apparently by Tigercub Software,
but no further information is known.
